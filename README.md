# 🎯 Face Recognition Based Attendance System

## 📌 Overview
This project is a **Biometric Attendance System** built using **Computer Vision and Face Recognition**.

It automatically detects and recognizes faces from a live webcam feed and marks attendance in a CSV file with timestamps.

---

## 🎯 Features
- 📸 Real-time face detection using webcam
- 🧠 Face recognition using pre-trained encodings
- 📝 Automatic attendance marking
- ⏱️ Records date and time of entry
- 📂 Stores attendance in CSV format
- 🚫 Prevents duplicate entries

---

## 🛠️ Tech Stack
- Python
- OpenCV
- face_recognition
- NumPy
